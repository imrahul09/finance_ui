# 💰 Finance Dashboard

A responsive and interactive Finance Dashboard built using React and Bootstrap. It allows users to track income, expenses, and gain insights into their financial activity.

---

## 🚀 Live Demo

👉 (financeui-theta.vercel.app) 

---

## 📌 Features

* 📊 Dashboard summary (Balance, Income, Expenses)
* 📋 Transactions table with filter & search
* 🔐 Role-based UI (Admin / Viewer)
* 📈 Charts (Line & Pie)
* 🧠 Insights section (spending analysis)
* 🌙 Dark mode support
* 💾 Data persistence using LocalStorage
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* React.js
* Bootstrap
* Recharts
* JavaScript (ES6)

---

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/finance-dashboard.git
```

2. Navigate to project folder

```bash
cd finance-dashboard
```

3. Install dependencies

```bash
npm install
```

4. Run the project

```bash
npm run dev
```
